# Consideraciones de accesibilidad

- Es necesario respetar las etiquetas semánticas, por lo que deben utilizarsen la etiqueta `<button>`.
- Deben respetarse los valores mínimos de contraste y tamaño mínimo en de la tipografía.
- Deben responder a la navegación e interacción con el teclado.

## Aria labels

La etiqueta de un botón puede estar escrita como `<button>` o proporcionada por el atributos `role="button"`.
Usar la propiedad `aria-disabled` para marcar como desabilitado el estado del botón.
