# Formatos de botones

Dependiendo el contexto, puede necesitarse que el botón se ajuste de forma diferente al contexto.
Estas son algunas de las transformaciones que pueden realizarse.

### ¿Cuándo usarlos?

- Los botones chicos se deben utilizar como acciones dentro de tablas o dentro de otros componentes que no tengan mucho espacio.
- Los botones grandes deben utilizarse cuando hay un único call to action en una página de marketing.
